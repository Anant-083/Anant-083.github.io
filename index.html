About Me

Building at the intersection of web, AI & civic tech

I'm a B.Tech Computer Science (AI & ML) student at Brainware University with a hands-on approach to learning — I build real things to understand how they work. My work spans full-stack web development, AI-powered applications, REST APIs, and machine learning.

I'm comfortable across the full stack: frontend interfaces in HTML, CSS & JavaScript, backends with Python, Flask, and FastAPI, integrations with LLMs via Groq API, database work in SQLite and SQL, and ML pipelines with scikit-learn and Pandas. I've also built desktop applications in Java with JavaFX.

Several of my projects are government-aligned civic tech tools — multilingual platforms for farmers and students, groundwater analytics dashboards, and resource allocation engines. I believe the most meaningful software solves problems for people who can't currently access the tools they need.

Every project I build is deployed to production and publicly available. I use Render, Vercel, Streamlit Cloud, and GitHub Pages to ship everything I write.
